# Icebound Outpost: The Frozen Trade Hub
## Overview
A harsh, snow-covered industrial outpost where only the outer coastal rim is exposed dirt—everything else lies beneath deep permafrost. The island survives not by self-sufficiency but by **mining frozen resources from beneath the ice** and **processing them for trade** with the main map. Its extreme climate makes it poor and dependent, constantly needing imports of food and supplies while exporting processed minerals and fuel to pay for survival. Two clear cargo flows: ore and fuel north, food and goods south.

---
### 🌍 Phase I: Local Resource Extraction (Frozen Wealth Below)
The island's value comes from mineral wealth hidden beneath the permafrost, plus accessible coastal gas seeps. Only the dirt rim allows basic surface access.

*   **Resource 1: Iron Ore Veins**: Dense deposits mined from snow-covered boreholes. *Location:* Northern Ice Fields (subsurface).
*   **Resource 2: Permafrost Methane**: Gas trapped in the frozen ground, venting at the southern dirt rim. *Location:* Southern Gas Seeps (dirt rim).
*   **Resource 3: Silica Gravel**: Purer-than-average sand found along the exposed coastal rim. *Location:* Coastal Strip (dirt rim).
*   **Resource 4: Limestone Shelf**: Accessible rocky outcrops along the western dirt shore. *Location:* Western Rocky Shore (dirt).

*Note: Everything north of the dirt rim is frozen solid. Mining requires drilling through permafrost.**

---
### ⚙️ Phase II: Processing (The Only Industry)
Raw frozen materials are heated, melted, and refined locally to create tradeable goods. All facilities sit on the dirt rim for access.

*   **From Iron Ore**: Smelted into **Steel Ingots** and **Iron Rebar**. *Facility:* Forge & Smelter (dirt rim, west).
*   **From Methane**: Bottled as **Fuel Canisters** for export. *Facility:* Gas Compression Station (dirt rim, south).
*   **From Silica Gravel**: Washed and processed into **Glass Sheets**. *Facility:* Glassworks (dirt rim, east).
*   **From Limestone**: Burned into **Quicklime** for cement and soil amendment. *Facility:* Lime Kiln (dirt rim, south).

*The dirt rim is sacred—all facilities cluster here where the ground is solid.*

---
### 📦 Phase III: Export (Paying for Survival)
The island's processed goods travel to the main map in exchange for everything it cannot grow or make.

*   **Primary Exports:**
    - **Steel Ingots** → Main Map factories (construction, vehicle manufacturing)
    - **Fuel Canisters** → Main Map gas stations and workshops
    - **Glass Sheets** → Main Map construction and vehicle windows
    - **Quicklime** → Main Map cement plants and farms (soil amendment)
*   **Export Route:** Dirt rim facilities → Loading Dock → Boat/Ferry to Main Map

---
### 🛒 Phase IV: Import (The Lifeline)
The island cannot grow food. Everything organic, plus machinery and luxuries, must be purchased through exports.

*   **Primary Imports:**
    - **Foodstuffs**: Bread, rice, canned meat, vegetables (main map farms)
    - **Parts & Machinery**: Drill bits, engine components, lubricant (main map factories)
    - **Construction Materials**: Standard lumber, concrete mix, rebar (main map bulk yards)
    - **Luxury Goods**: Cigarettes, coffee, alcohol, warm clothing (main map shops)
    - **Chemicals**: Flux agents, solvents for processing (main map chemical plants)

*   **Import Route:** Main Map → Boat/Ferry → Loading Dock → Central Warehouse → Distribution

---
### 🔄 Cargo Flow Summary: The Vital Exchange
**Outbound (Island → Main Map):**
1.  Northern Ice Fields → Drill → Iron Ore → Forge & Smelter → Steel Ingots → Loading Dock → Main Map
2.  Southern Gas Seeps → Compression → Fuel Canisters → Loading Dock → Main Map Gas Stations
3.  Coastal Strip → Glassworks → Glass Sheets → Loading Dock → Main Map Construction
4.  Western Shore → Lime Kiln → Quicklime → Loading Dock → Main Map Farms/Cement Plants

**Inbound (Main Map → Island):**
1.  Main Map Farms → Food Shipment → Island Food Warehouse → Residential Zones
2.  Main Map Factories → Parts Shipment → Island Maintenance Shop
3.  Main Map Shops → Goods Shipment → Island Retail Store
4.  Main Map Bulk Yards → Materials Shipment → Island Construction Sites
5.  Main Map Chemical Plants → Chemicals Shipment → Island Processing Plants

**Key Insight:** The island is poor. Profit margins are thin. One bad season—one collapsed export price, one missed shipment—and the outpost struggles. This creates constant, high-stakes gameplay around maintaining the cargo flow.
